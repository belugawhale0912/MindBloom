## 📖 Overview

**MindBloom** is a web-based mental wellness platform designed to be a calm, accessible companion for everyday mental health. From tracking your mood to guided breathing exercises, MindBloom offers a suite of gentle tools that meet you where you are — no pressure, just support.

> Built as a submission for **NAIC 2026**, MindBloom combines evidence-informed wellness techniques with an approachable, modern interface.

---

## ✨ Features

| Feature | Description |
|---|---|
| 🏠 **Dashboard** | Personalized welcome, mood quick-check, daily quotes, and streak tracking |
| 😊 **Mood Tracker** | Log how you feel throughout the day and spot emotional patterns over time |
| 🤖 **AI Companion** | A live chat-based companion (powered by Botpress) for when you need someone to talk to |
| 🧘 **Guided Tools** | Curated exercises to help regulate your nervous system and ground yourself |
| 🆘 **Crisis Support** | Immediate access to crisis resources — always one tap away |
| ⏰ **Reminders** | Set gentle check-in reminders to build a consistent self-care routine |
| 📈 **Insights** | Visual mood trends and progress tracking over time |
| 📋 **Self-Assessment** | Evidence-based mental health check-ins to gauge your wellbeing |
| 🧺 **My Calm Kit** | A personal sanctuary of your go-to coping tools and resources |
| 🎵 **White Noise Mixer** | Mix and layer ambient sounds to create your perfect focus or relaxation environment |
| 🌿 **Zen Space** | A distraction-free space for stillness and calm |

## 🛠️ Tech Stack

- **Framework:** [Next.js](https://nextjs.org/) (React)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/) + [shadcn/ui](https://ui.shadcn.com/)
- **AI Companion:** [Botpress](https://botpress.com/) (live chatbot integration)
- **Deployment:** [Vercel](https://vercel.com/)

## 🚀 Getting Started

### Prerequisites

- Node.js `v18+`
- npm or pnpm

### Installation

```bash
# Clone the repository
git clone https://github.com/belugawhale0912/mindbloom.git
cd mindbloom

# Install dependencies
npm install

# Start the development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build for Production

```bash
npm run build
npm start
```
